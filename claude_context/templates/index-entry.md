# INDEX.md Entry Template

Use this format when adding rows to `claude_context/INDEX.md`.

## For Knowledge Files (product/, technical/, audits/, domain/)

```markdown
| path/FILE.md | ~NNN | One-line summary | When to read this file |
```

## For Reports

```markdown
| reports/YYYY-MM-DD-topic.md | YYYY-MM-DD | Topic | Key finding summary |
```

## Token Estimation

Estimate token count: ~4 characters per token, or count words and multiply by 1.3.

Always update the token count when a file changes significantly.
