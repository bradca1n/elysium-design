# iPhone 16 Mockup Plugin — Recommendation

## Winner: iPhone 16 & 17 Free Mockup (Community File)

**Link:** https://www.figma.com/community/file/1064512555184175821/iphone-16-17-free-mockup

### Why This One

| Criteria | Score |
|---|---|
| Ease of use | Paste component into screen frame — done |
| Free | Yes, no restrictions |
| Vector-native | 100% vector, built in Figma |
| iPhone 16 Pro | Covers iPhone 16 + 17 models |
| Live components | Screen content stays editable (not rasterized) |
| Maintained | Already updated for iPhone 17 |
| Customizable | All colors, hideable lens/antenna details |

### How to Use for Component Documentation

1. Duplicate the community file into your Figma account
2. Copy the device frame into your Product-Demo file (Base Components page or a dedicated "Device Previews" section)
3. For each component you want to document:
   - Create a component instance
   - Place it inside the device screen frame
   - The component stays live and editable

### Alternatives Considered

| Option | Type | Free | Why Not |
|---|---|---|---|
| Mockuuups Studio | Plugin | Freemium ($12/mo) | Exports rasterized PNGs, not live vector frames. Defeats documentation purpose. |
| Design Monks iPhone 16/Pro | Community file | Yes | Good but less documented, fewer models |
| iPhone 16 UI Mockups | Community file | Yes | Least documented, bundled with sample UI |
